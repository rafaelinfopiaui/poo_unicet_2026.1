# Programação Orientada a Objetos em Java ☕

Este repositório contém as soluções das atividades de Programação Orientada a Objetos (POO), desenvolvidas durante o curso no **UNI-CET**. O projeto está organizado para refletir a evolução do aprendizado ao longo das semanas, focando na transição do paradigma procedimental para o orientado a objetos.

## 🛠️ Especificações Técnicas

- **Autor:** Rafael Sampaio Oliveira (Graduando em Engenharia de Computação)
- **Linguagem:** Java (OpenJDK 11+)
- **Estrutura:** Organizado por Semanas e Atividades Práticas.

---

## 🏗️ Estrutura do Repositório

O repositório está organizado em pastas por semana de atividade, permitindo uma navegação clara pela evolução do conteúdo:

### 📂 Atividade_Pratica_Semana_1/

Conteúdo focado nos fundamentos de POO, abstração e encapsulamento.

- **Bloco_01: Fundamentos e Abstração**
  - Classes como moldes, gerenciamento de memória (Pessoa, Carro, Conta Bancária V1, Lâmpada).
- **Bloco_02: Comportamento e Encapsulamento**
  - Métodos, parâmetros, retornos e composição de objetos (Produto, Paciente, Funcionário, Pessoa & Endereço).
- **Bloco_03: Lógica Avançada e Interatividade**
  - Estruturas de repetição, validação com Setters, arrays de objetos e interatividade com `Scanner` (Termômetro, Estudante, Sistema de Login, Agente de Saúde).
- **exemplos/**: Exemplos diversos utilizados durante as aulas.

---

## 🚀 Conceitos de Engenharia Aplicados

1.  **Encapsulamento:** Uso de modificadores `private` e métodos acessores (Getters/Setters).
2.  **Abstração:** Modelagem de entidades do mundo real em classes coesas.
3.  **Composição:** Criação de objetos complexos a partir de outros objetos.
4.  **Higiene de Código:** Código limpo, comentado e organizado.

## 🔧 Como Executar

Para rodar os exercícios localmente:

1. Clone o repositório:

   ```bash
   git clone https://github.com/rafaelinfopiaui/poo_unicet_2026.1.git
   ```

2. Entre no diretório da semana e do bloco desejado:

   ```bash
   cd Atividade_Pratica_Semana_1/Bloco_03
   ```

3. Compile os arquivos Java:

   ```bash
   javac *.java
   ```

4. Execute a classe de teste principal:
   ```bash
   java TesteAgenteSaude
   ```
